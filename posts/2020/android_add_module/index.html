【Android】モジュール追加 私はこんな風にしてます

2020/5/27 1-minute read
Android

Androidでモジュールを追加するとき。 例えば

modules/features/example/ui

というディレクトリのexample-uiモジュールを作りたいとき。

AndroidStudioの左側Projectのあたりで右クリックし

New > Module

するとダイアログがでてくるので

Android Library を選択し

Application/Library name → example-ui  
Module name → :modules:features:example:ui  
package name → com.hogehoge.example.ui

としてFinishすることで完了。
